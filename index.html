Ultimate Assault

ULTIMATE ASSAULT is a multiplayer 2D Third Person Shooter Team Deathmatch game. Most of the mechanics is inspired by the Call of Duty games series and are applied into a 2D game world.

The game is still in development mode and the parameters are not well calibrated. Players may also experience errors and instabilities.

Credits for the graphics go to,

It is strongly recommended to visit Features for information about the implemented options details.

How To Play

  1. Connect to your local network (Every player must connect)
  2. One of the players creates a game setting the number of the players; hence setting up the server
    (Map and Game Time are set by the player)
  3. Others join the created game using the local IP address of the server
  4. Every player selects his/her name (Should be distinct), Class and team
  5. Players select their Weapons (Compatibility is illustrated by colors) and Tactical Grenades
  6. Each player selects 3 Perk options (Including no-perk option) and 3 Killstreaks
  7. Game starts when all the players complete their setup
To run the game on a single machine for test purposes : You can run multiple instances of the game on the same machine, create a game as the server (host) in one of the instances and set the local IP of the machie for other instances (clients) to join.

Controls

W, S, A, D and Mouse pointer : Move and strafe
Mouse left and right : Shoot and aim
Mouse middle : Lethal grenade toss
Shift : Hold to sprint until no stamina
Q : Tactical grenade toss
E : Melee attack
F : Press to open airdrops/ Press to mount and unmount turrets/ Hold to defuse mines/ Hold to capture machines
Shift + F : Hold to trap and de-trap boxes
1, 2, 3 : Use killstreaks
Tab : Hold to see player map and info
F4 : Toggle the fullscreen
