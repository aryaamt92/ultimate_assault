Classes

Killstreaks
Killstreaks are activated for the player by achieving kills in row (Use 1, 2 and 3 keys). There is a cooldown time after the killstreak is used. Change the path selector direction using mouse wheel.
UAV Radar - 3 Kills

Reveals the enemies positions to the team players on map for a limited amount on time
Recon : Enemies directions are also revealed

Counter UAV - 3 Kills

Blocks enemy UAV from revealing player's position for a limited amount of time
Airdrop Package - 4 Kills

Calls for an airdrop ammo package to the selected position by player
Electro-Magnetic Pulse - 5 Kills

Calls for an EMP shutting down enemy radars, laser aims, scopes and thermal visions for a certain amount of time
Coyote Suicide Drone - 5 Kills

Calls for a Coyote suicide drone to be controlled and detonated by player
Drone is detonated automaticall after a certain amount of time
Sentry Gun - 6 Kills

Player can install an automatic Sentry gun
Shoots enemies down, can be damaged, destroyed and captured by engineers
Precision Strike - 6 Kills

Calls for 3 precision air strikes on the selected position via map
Pro Hard-line : Precision Napalm Strike
Artillary Strike - 6 Kills

Calls for 3 rounds of artillary strikes on the selected position via laser beacon
Predator Missile - 7 Kills

Activates a Predator UAV for a single guided missile strike
Use mouse right to switch between vision modes
Heli Guard - 7 Kills

Calls for an unmanned light helicopter to follow and protect the player
(Not active yet)
Harrier Strike - 7 Kills

Calls for a hovering Harrier to cover the selected area via map for a limited amount of time
Air Supply - 8 Kills

Calls for three packages to be dropped along the selected path via map
Packages include ammo, weapon, personal radar, UGCV, advanced UAV recon and MG turret
Carpet Bomber - 9 Kills

Calls for a carpet bombing along the selected path via map
Pro Hard-line : Carpet Napalm Bomber
Chopper Gunner - 10 Kills

Calls for a attack helicopter to be controlled by the player for a limited amount of time
(Not active yet)
Pave Low - 11 Kills

Calls for a heavy attack helicopter to cover the selected area via map for a limited amount of time
(Not active yet)
White Phosphorus - 12 Kills

Calls for a white phosphorus attack along the selected path via map weakening enemies
AC-130 Gunship - 12 Kills

Calls for a gunship to be controlled by the player for a limited amount of time
Use mouse wheel to switch between cannon and machine gun
Use mouse right to switch between vision modes
Tactical Nuke - 15 Kills

Calls for a tactical nuclear strike claiming victory and ending the game right away
